# Triple Peaks Coffee Shop

This project is a landing page for the **Triple Peaks Coffee Shop**, created as part of the Software Engineering program at TripleTen. The webpage was built using modern HTML and CSS practices based on the provided design brief.

It presents information about the coffee shop and includes a custom reservation form, with a focus on clean structure, maintainable code, and user-friendly styling.

## Project Features

- Semantic HTML5 for clear structure and accessibility
- CSS3 styling with Flexbox and positioning
- Flat BEM file structure for scalable and maintainable code
- Custom reservation form with required inputs
- CSS animations, transforms, and hover effects
- Normalize.css for consistent styling across browsers

## Technologies Used

- HTML5
- CSS3
- BEM methodology
- Flexbox
- Git & GitHub

## Plans for Improvement

In future iterations of this project, I would like to:
- Make the webpage fully responsive for mobile and tablet devices
- Add JavaScript form validation and user feedback
- Improve accessibility by enhancing contrast and adding ARIA attributes
- Add subtle scroll-based animations to improve user experience
- Connect the reservation form to a backend service or API for real submissions

These improvements would help make the project more interactive, accessible, and closer to a real-world production website.
